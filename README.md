# 🏥 Intelligence in Health Care

## 📌 Project Overview

**Intelligence in Health Care** is a web-based healthcare prediction system developed using **HTML, CSS, and JavaScript**. The application allows users to enter basic health parameters and provides a simple health risk analysis based on predefined conditions.

> **Note:** This project is developed for educational and demonstration purposes only. It is not intended for real medical diagnosis.

---

## ✨ Features

- Attractive and responsive user interface
- Visitor counter using Local Storage
- Rotating health tips
- Patient information form
- Automatic BMI calculation
- Health risk prediction
- Risk percentage meter
- Health analysis report
- Personalized health recommendations
- Home page and health status page navigation

---

## 🩺 Input Parameters

The application accepts the following health details:

- Patient Name
- Gender
- Age
- Height (cm)
- Weight (kg)
- Blood Pressure
- Blood Sugar
- Cholesterol
- Heart Rate

---

## 📊 Health Analysis

The system performs:

- BMI Calculation
- Risk Score Calculation
- Disease Prediction
- Health Risk Classification
- Blood Sugar Estimation
- Similar Patient Analysis
- Diagnosis Explanation
- Patient Category Analysis
- Key Health Factors Identification

---

## 🧠 Algorithms Demonstrated

The project demonstrates the concepts of the following machine learning algorithms:

- Linear Regression (Blood Sugar Estimation)
- Logistic Regression (Disease Prediction)
- K-Nearest Neighbors (Similar Patient Analysis)
- Decision Tree (Diagnosis Explanation)
- Random Forest (Health Risk Classification)
- K-Means Clustering (Patient Category)
- Principal Component Analysis (Key Health Factors)

> These algorithms are represented for educational purposes using JavaScript logic and are not trained machine learning models.

## 🚀 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click **health.html** or open it in any modern web browser.
4. Enter patient details.
5. Click **Submit** to view the health prediction.
